## Chat App

Project is created with:

- React
- TypeScript
- Context API
- Firebase (firestore & auth)

## Watch live

This project is deployed on Firebase Hosting.
[Click here](https://chat-app-69545.web.app/) to watch it live.

## Setup

To run this project, install it locally using npm:

```
cd chat-app
npm install
npm start
```
